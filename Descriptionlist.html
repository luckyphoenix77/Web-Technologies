Web Technologies

HTML
-HyperText Markup Language
-The foundation of the web, HTML structures content with headings, paragraphs, lists, links, images, and more. It defines the semantic meaning of a document.
CSS
-Cascading Style Sheets
-Responsible for the presentation of HTML content, CSS styles how elements are displayed, including layout, colors, fonts, and responsiveness.
JavaScript
-A scripting language that adds interactivity, dynamic behavior, and rich user experiences to web pages. It can manipulate the Document Object Model (DOM), handle events, and communicate with servers.
PHP
-Hypertext Preprocessor
-A widely-used scripting language primarily for web development. It's embedded in HTML to create dynamic content and interact with databases.
